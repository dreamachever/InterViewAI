面向保研场景的 AI 模拟面试系统，支持简历上传与诊断、文字问答、AI 面试官语音播报、用户语音回答转文字、结构化评分报告和用户自定义大模型配置。

项目主体位于 `ai-interview/`：

- `frontend/`：React + TypeScript + Vite + Ant Design
- `backend/`：FastAPI + SQLAlchemy + PostgreSQL + Alembic

详细启动方式见 `ai-interview/README.md`。
=======
面向保研面试的文字版 AI 模拟面试应用。初步功能是用户填写面试配置并粘贴简历文本后，系统会生成专属 AI 面试官，进行多轮文字追问，并在结束后生成结构化评分报告。后续会陆续加上语音面试、上传和解析pdf简历等功能。
